# Council Point

> Hillingdon x Brunel University Hackathon

**Team:** The Ducks
> Awarded the 2nd place in the hackathon and chosen as the best project by Hillingdon Council.

Council Point is a document management system that allows council workers to upload hand written or digital files. It then processes these documents to translate then to english (if required), redacted sensitive information before sharing to others, and calculating a GDPR compliance score. It provides a user-friendly interface for managing documents, including document search feature which can be used to facilitate GDPR Subject Access Requests (SAR). The system is designed to be accessible to all users and includes features such as text size adjustment, dark mode, and language selection.
